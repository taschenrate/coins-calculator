# Калькулятор коинов модераторов

Статический мини-дашборд для расчета наград модераторов по количеству проверок. Работает без сервера: все записи сохраняются в LocalStorage браузера.

## Что умеет сайт

- рассчитывает коины по заданным правилам начисления;
- сохраняет, обновляет, редактирует и удаляет модераторов;
- ищет по нику и сортирует таблицу;
- показывает статистику и топ-5 на графике;
- экспортирует базу в TXT и CSV;
- импортирует данные из TXT и CSV с пересчетом коинов;
- очищает локальную базу после подтверждения.
